# Laravel-CMS

## Description  
This CMS is using Laravel that was made by basic system.

### function list  
- CRUD
- File Upload

## Install  
```
$ git clone https://github.com/Restoration/Laravel-CMS.git
$ cd Laravel-CMS/app
$ vim .env # setting environment, connect database
$ php artisan migrate
```
